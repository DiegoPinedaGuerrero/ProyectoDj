Datos clave antes de cotizar

¿Cuántas personas son?

El precio base cubre de 10 a 100 personas. Para 100-200 se agregan $3,000, para 200-300 se agregan $5,500 y para 300 o más se agregan $7,500.

¿El evento es en espacio abierto o cerrado?

El tipo de espacio afecta la cantidad de audio, iluminación y montaje. El formulario permite seleccionar interior o exterior.

¿Qué tipo de evento es?

Se puede configurar para boda, cumpleaños, privado, pool party, hotel o corporativo.

¿Cómo aparto la fecha?

El anticipo sugerido es de $1,500 MXN. La pasarela de pago puede conectarse con Mercado Pago.

¿Trabajan fuera de CDMX?

Sí, pero para interior de la república se cotiza por WhatsApp según distancia, logística y requerimientos técnicos.
